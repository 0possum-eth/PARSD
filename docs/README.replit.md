# PARSD on Replit (Skills-Only)

PARSD on Replit installs the skill corpus into your project:

- `./.agents/skills/*`

## Recommended Install Path

From chat, tell Replit:

```text
Fetch and follow instructions from https://raw.githubusercontent.com/0possum-eth/PARSD/refs/heads/main/.replit/INSTALL.md
```

## What Gets Installed

- Arbiter/SuperDAO skills
- Supporting Superpowers-compatible workflow skills

## What Does Not Get Installed in Replit Mode

- OpenCode plugin hooks
- Codex/OpenCode runtime file wiring for automatic M1-M5 execution

Use Codex/OpenCode installs when you need automatic runtime hook behavior.
